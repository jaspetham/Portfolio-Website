FYP Quiz App

This will be the showcase of my final year Quiz App.

Introduction

FYP Quiz App is a final year project for my course. The app is based on the how to be a millionaire game so i decided to build this in android studio.

Functions

To start it off, I will be using Firebase database for my main storage of data. This include my questions, answer, users, ranking, scores. See image above.

Login/Registration - The app will have login function and registration to make the app more complex. In addition, the sign up will be needed. all this data that user filled in will be added into the firebase database and is updated in real-time. It also check for any existing user if there isn't, it will be add the new user.

Main Page - Once logged in, there will be 2 tabs. First is the categories of question for the user to choose to play. Once chosen a category, there will be a play button to start the game.

Gameplay - The gameplay will be in 10 questions for each category, there will be 2 kind of question. Text based and image based question. There will be a progress bar to display the time left for each question and score will be tracked and which question the user is in. After the end of the question, there will be a summary of your question score.

Scoreboard - In the main page, there will be a scoreboard. This scoreboard track all of the user total combine score of all the categories. The user's name can be tapped to show their score of each categories. The score will be updated in real-time with the help of firebase.
